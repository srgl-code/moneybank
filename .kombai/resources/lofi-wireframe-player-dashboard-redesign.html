📝 Notas do Redesign

Mudanças Principais:

  • Textos 100% em PT-BR — "Patrimônio Total" ao invés de "Total Wealth", "Transferir" ao invés de "Initiate Transfer"
  • Novos StatCards — Posição no ranking, total de transações, tendência visíveis de relance
  • Room Code Banner — Código da sala visível para o jogador compartilhar facilmente
  • Pending Banner — Status de transferências pendentes de aprovação
  • Tabs no conteúdo — Extrato e Ranking como abas inline ao invés de depender apenas do bottom nav
  • Bottom Nav simplificado — Apenas 2 abas (Carteira/Ranking) com labels em PT-BR
  • Landmarks acessíveis — main, nav, section com roles apropriados

Componentes Reutilizados (5):

  • AppHeader — Cabeçalho com logo e botão de sair
  • AnimatedBalance — Saldo animado com direção
  • TransferModal — Modal de transferência (aberto pelo CTA)
  • TransactionHistory — Lista de transações
  • BottomNav — Navegação inferior mobile

Novos Componentes (4):

  • RoomCodeBanner — Mostra código da sala com botão copiar
  • StatCards — Cards resumo (posição, transações, tendência)
  • ContentTabs — Abas inline para Extrato/Ranking
  • PendingBanner — Alerta de transferências pendentes
