📝 Notas do Redesign — Home

  • Hero animado — Logo com bounce, partículas flutuantes, gradiente pulsante no fundo
  • 3 abas — Entrar / Criar / Recentes (nova aba para reconexão rápida)
  • Pinos maiores — Grid com labels de cor, ring animado na seleção, melhor acessibilidade (aria-labels)
  • Modos de jogo — Cards visuais substituem preset simples (Rápido/Clássico/Luxo)
  • Salas recentes — localStorage para rejoin rápido sem digitar código
  • 100% PT-BR — Todos os textos em português
  • Botão CTA — Gradiente vibrante com efeito shine sweep e sombra 3D
  • Checkbox acessível — "Outro valor" usa role="checkbox" + aria-checked
